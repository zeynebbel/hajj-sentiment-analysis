# Hajj Sentiment Analysis

An AI-powered web application built with **Streamlit** to analyze Hajj and Umrah comments using Natural Language Processing (NLP) and Machine Learning.

The application helps identify user sentiment, detect emotions, highlight potential issues, and generate professional reports for decision-making.

---

##  Features

-  Analyze a single comment
-  Analyze multiple comments from a CSV file
-  Sentiment classification (Positive, Neutral, Negative)
-  Emotion detection
-  Problem detection in negative comments
- Interactive visualizations and statistics
- Word Cloud generation
- Automatic PDF report generation
-  User-friendly Streamlit interface

---

##  Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- WordCloud
- Machine Learning
- Natural Language Processing (NLP)

##  Installation

Clone the repository:

```bash
git clone https://github.com/zeynebbel/hajj-sentiment-analysis.git
```

Go to the project folder:

```bash
cd hajj-sentiment-analysis
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
